# 💻Include new Addon

    {
        "name": "MasterMount",
        "githubRepo": "https://github.com/PentSec/MasterMount", --Link repo
        "imageUrl": "https://wow.zamimg.com/images/wow/icons/large/inv_mount_spectralhorse.jpg", --a logo referent to addon
